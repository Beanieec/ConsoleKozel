### You must download this these dependencies:
>CPR

>nlohman/json


### VCPKG installing

```
git clone https://github.com/Microsoft/vcpkg.git

cd "your way"/vcpkg

.\bootstrap-vcpkg.bat

.\vcpkg integrate install

```
```
.\vcpkg install cpr
```
```
.\vcpkg install nlohmann-json
```
